## Project Overview
This project focuses on Exploratory Data Analysis (EDA) of Uber ride data to understand
booking patterns, trip duration, distance, and fare behavior.

## Dataset
- Pickup & drop locations
- Trip distance and duration
- Fare amount
- Booking status

## Objectives
- Analyze booking trends
- Study fare distribution
- Understand relationship between distance, duration, and fare
- Detect outliers and anomalies

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Insights
- Fare increases with trip distance and duration
- Peak booking times observed during specific hours
- Presence of fare outliers for long-distance trips

## Visualizations
- Histograms
- Box plots
- Scatter plots
- Correlation heatmap

## Conclusion
EDA helped uncover meaningful insights that can support pricing optimization
and demand forecasting.
